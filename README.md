# Java GUI Calculator

A simple calculator application built in Java using Swing.

## Features
- Addition, subtraction, multiplication, and division
- Supports decimal inputs
- Clear and delete functionality
- Graphical interface using JFrame, JButton, and JTextField

## Technologies Used
- Java
- Swing (AWT/Swing components)

## How to Run
1. Compile the program:
   javac Calc.java
2. Run the program:
   java Calc
